🥤 SlurpeeMod for 7 Days to Die
A custom modlet that introduces the Kirk's Slurpee—a high-tier beverage designed for the ultimate scavenger. It provides a significant boost to your loot stage but carries the tactical risk of a brain freeze.

🚀 Features
Custom 3D Model: High-quality Unity asset integration for the Slurpee cup.

Loot Boost: Grants the Slurpee Luck buff, increasing your Loot Stage by +15.

Risk/Reward: Includes a custom Brain Freeze debuff that temporarily slows movement.

Custom Icons: Unique UI icons for both the item and the associated buffs.

🛠 Installation (Client & Server Required)
IMPORTANT: This mod contains custom 3D models and UI assets. It MUST be installed on both the server and by every player joining the server.

Download the latest release.

Extract the folder into your 7 Days To Die/Mods directory.

Ensure the folder is named z_SlurpeeMod to ensure it overrides any conflicting UI mods (like XRamosZUI).

📂 Technical Details
Version: 5.2.1.1

Pathing: Uses #@modfolder dynamic pathing for maximum asset reliability.
